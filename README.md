# OpenCV 01주차 실습

본 프로젝트는 OpenCV 라이브러리를 활용한 실습 과제들을 포함하고 있습니다.

## 🛠 환경 설정 (Environment)
*   **Python**: 3.12.x
*   **Library**: `opencv-python`, `numpy`
*   **IDE**: `Antigravity` (Agentic AI Coding Assistant)

---

## 📂 파일별 상세 기능 설명

### 1. 이미지 불러오기 및 그레이스케일 변환 (`1.py`)
이미지를 불러와서 기초적인 기하학적/색상 변환을 수행하고, 두 가지 다른 상태의 이미지를 하나로 합쳐서 보여주는 실습입니다.
*   **주요 기능**:
    *   `cv.imread()`를 활용하여 이미지 로드 및 절대 경로 처리
    *   `cv.resize()`를 사용하여 이미지 사이즈를 50%로 축소 (처리 속도 및 화면 배치 최적화)
    *   `cv.cvtColor()`를 통해 원본(BGR) 이미지를 그레이스케일(Grayscale)로 변환
    *   `np.hstack()` 기능을 활용하여 원본 이미지와 흑백 이미지를 가로로 나란히 이어 붙여 비교 출력
    *   `cv.imwrite()`를 통해 최종 결합된 결과물을 `1_result.jpg` 파일로 자동 저장

### 2. 페인팅 붓 크기 조절 기능 추가 (`2.py`)
마우스와 키보드 이벤트를 결합하여 이미지 위에 자유롭게 그림을 그릴 수 있는 인터랙티브 페인팅 툴 실습입니다.
*   **주요 기능**:
    *   **마우스 이벤트**: 좌클릭(파란색), 우클릭(빨간색), 드래그(연속 그리기) 기능을 지원합니다.
    *   **실시간 크기 조절**: 키보드의 `+` 키와 `-` 키를 사용하여 붓의 크기를 1부터 15까지 실시간으로 조절 가능합니다.
    *   **시각화**: 터미널 창을 통해 현재 적용된 붓의 크기를 숫자로 실시간 출력하여 사용자가 인지하도록 돕습니다.
    *   **자동 저장**: `q` 키를 눌러 종료하면 수정한 이미지가 `2_result.jpg`로 즉시 저장됩니다.

### 3. 마우스로 영역 선택 및 ROI(관심영역) 추출 (`3.py`)
이미지의 특정 부분만을 선택하고 정밀하게 잘라내어 별도의 결과물로 만드는 실습입니다.
*   **주요 기능**:
    *   **드래그 가이드**: 마우스 왼쪽 버튼을 누른 채 드래그하는 동안 실시간으로 초록색 가이드 사각형을 그려주어 영역 선택 가독성을 높였습니다.
    *   **영역 확정 및 추출**: 마우스 버튼을 떼는 순간 선택된 부분을 Numpy 슬라이싱 기법으로 정교하게 잘라내어 별도의 창('Selected ROI')에 즉시 띄웁니다.
    *   **좌표 자동 정렬**: 상하좌우 어느 방향으로 드래그해도 `min/max` 연산을 통해 올바른 사각형 영역을 계산하도록 구현되었습니다.
    *   **기능키 지원**: `r` 키를 눌러 선택 영역을 리셋하거나, `s` 키를 눌러 최종 선택된 영역만 `3_result.jpg` 파일로 분리 저장할 수 있습니다.

---

## ⌨️ 공통 조작키 안내
*   `q`: 프로그램 종료 및 (지원하는 경우) 결과 저장
*   `r`: (3.py 전용) 영역 선택 초기화
*   `s`: (3.py 전용) 현재 선택된 ROI 이미지 저장
*   `+`, `-`: (2.py 전용) 붓 크기 조절
