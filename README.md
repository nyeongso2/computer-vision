# OpenCV 01주차 실습

본 프로젝트는 OpenCV 라이브러리를 활용한 기초 실습 과제들을 포함하고 있습니다. 모든 코드는 가독성을 위해 각 줄마다 상세한 주석이 작성되어 있습니다.

## 🛠 환경 설정 (Environment)
*   **Python**: 3.11.9
*   **Library**: `opencv-python`, `numpy`
*   **IDE**: `Antigravity` (Agentic AI Coding Assistant)

---

## 📂 과제별 상세 설명 및 결과

### 01. 이미지 불러오기 및 그레이스케일 변환
이미지를 불러와서 기초적인 기하학적/색상 변환을 수행하고, 원본과 변환된 이미지를 나란히 비교 출력합니다.
*   **주요 기능**: `cv.imread()`, `cv.cvtColor()`, `np.hstack()` 활용
*   **코드 확인**: [1.py](./opencv/01주차%20과제/1.py) (모든 라인 주석 완료)
*   **최종 결과물 (`1_result.jpg`)**:
    ![1번 결과](./opencv/01주차%20과제/1_result.jpg)

---

### 02. 페인팅 붓 크기 조절 기능 추가
마우스와 키보드 이벤트를 결합하여 이미지 위에 실시간으로 그림을 그리는 인터랙티브 툴입니다.
*   **주요 기능**: 좌/우클릭 색상 변경, `+`/`-` 키를 이용한 붓 크기(1~15) 조절
*   **코드 확인**: [2.py](./opencv/01주차%20과제/2.py) (모든 라인 주석 완료)
*   **최종 결과물 (`2_result.jpg`)**:
    ![2번 결과](./opencv/01주차%20과제/2_result.jpg)

---

### 03. 마우스로 영역 선택 및 ROI(관심영역) 추출
드래그를 통해 이미지의 특정 영역을 선택하고, 해당 부분을 별도의 이미지로 추출 및 저장합니다.
*   **주요 기능**: 실시간 드래그 가이드 사각형, ROI 슬라이싱 추출, `r`(리셋), `s`(저장)
*   **코드 확인**: [3.py](./opencv/01주차%20과제/3.py) (모든 라인 주석 완료)
*   **최종 결과물 (`3_result.jpg`)**:
    ![3번 결과](./opencv/01주차%20과제/3_result.jpg)

## ⌨️ 공통 조작키 안내
*   `q`: 프로그램 종료 및 결과 저장
*   `r`: (3.py 전용) 영역 선택 초기화
*   `s`: (3.py 전용) 현재 선택된 ROI 이미지 저장
*   `+`, `-`: (2.py 전용) 붓 크기 조절
